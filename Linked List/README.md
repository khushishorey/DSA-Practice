## Linked Lists

This folder contains C++ implementations and problems related to
**singly and doubly linked lists**.

### Concepts Covered
- Node structure and pointers
- Insertion and deletion
- Reversal of linked list
- Fast and slow pointer technique
- Cycle detection

### Focus Areas
- Pointer manipulation
- Memory management basics
- Avoiding common pointer-related bugs

These problems are solved to strengthen understanding of **dynamic data structures**
and pointer logic in C++.
